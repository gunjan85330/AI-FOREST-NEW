# AI-Forest  
Wildfire Prediction & Spread Simulation System

## Overview
AI-Forest is an AI-driven wildfire analysis platform that predicts fire risk, simulates spread, and visualizes vulnerability across Uttarakhand using NASA FIRMS and ERA5 weather data.

## Features
-  Fire risk prediction  
-  Wildfire spread simulation  
-  Heatmap visualizations  
-  Project overview  
-  Contributors page  
-  Login protected dashboard  

## Tech Stack
- Python  
- Streamlit  
- XGBoost  
- ERA5 Weather Data  
- NASA VIIRS Fire Data  
- KDTree Nearest Neighbor Search  

## Deployment
Deployed using Streamlit Cloud.
